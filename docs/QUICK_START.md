# Quick Start Guide

Get your Fedora system set up in 15 minutes.

## Prerequisites

- Fedora 42+ freshly installed
- Internet connection
- sudo access

## Setup

### Step 1: Clone the Guide
```bash
git clone https://github.com/neardaniel-pls/fedora-system-setup.git
cd fedora-system-setup
```

### Step 2: Open the Guide
Read [`setup/fedora_initial_setup.md`](../setup/fedora_initial_setup.md) and follow the steps.

### Step 3: Follow In Order
The guide is organized by priority:
1. System updates
2. Repositories (RPM Fusion)
3. Flatpak/Flathub
4. Multimedia codecs
5. GNOME extensions
6. System customization
