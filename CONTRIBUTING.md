# Contributing

1. Fork the repo at https://github.com/neardaniel-pls/fedora-system-setup
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Push and open a pull request

## Guide Guidelines

- Test all commands on a fresh Fedora installation
- Include prerequisites at the top of each section
- Use code blocks for all terminal commands
- Keep guides focused on one topic
- Note any version-specific instructions

## Commit Messages

Use conventional commits: `feat`, `fix`, `docs`, `refactor`, `chore`
